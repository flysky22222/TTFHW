# 昇腾领域主导开源社区TTFHW场景用例

## CANN

### 贡献者场景
作为一个CANN的开发者，在Linux服务器上完成一个新的CANN算子的开发和验证,具体操作步骤如下：
1. 在Baidu及豆包搜索平台中搜索CANN的相关信息
2. 根据搜索到的信息进行CANN的下载和本地编译
3. 对ops-math仓库中的算子进行编译和测试
4. 根据Readme和开发指导，完成Floor算子的开发
5. 对开发完成的算子进行编译和运行测试
6. 将开发好的算子贡献到CANN上游代码仓库中

### 用户场景
作为一个CANN的用户，在昇腾A3单节点上使用CANN，成功运行一个模型推理，具体的操作步骤如下所示：
1. 在Baidu及豆包搜索平台中搜索CANN的相关信息
2. 根据搜索到的信息进行CANN的下载和安装，使用Readme中推荐的OS
3. 根据社区提供的quick start或者readme完成Qwen3.5-8B模型推理场景的跑通
